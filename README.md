## 📋 About

This is a modern, responsive portfolio website for **Mohamed Abdeen**, a Business Intelligence & Machine Learning Engineer. The portfolio showcases professional experience, technical skills, projects, and services.

**Unique Selling Point:** *"I don't just work with data - I solve business problems with it."*

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth scrolling navigation
- Mobile-friendly hamburger menu
- Interactive project cards with hover effects
- Service cards showcasing professional offerings
- Contact information and social media integration
- Downloadable CV
- Clean, modern UI with blue accent colors

## 🛠️ Technologies Used

- **HTML5** - Structure
- **CSS3** - Styling and animations
- **JavaScript** - Interactivity and mobile menu
- **Font Awesome 6** - Icons
- **Google Fonts** - Poppins & Roboto

## 📑 Sections

1. **Home** - Introduction with profile image
2. **About** - Personal statement and USP
3. **Experience** - Work history and internships
4. **Education** - Academic background
5. **Services** - Professional services offered
6. **Projects** - Showcase of data projects
7. **Skills** - Technical skills categorized
8. **Contact** - Contact information
